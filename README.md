# 🥔 Amazing Click Gaming

> "Prepare-se para as melhores mecânicas do mundo!"

O **Amazing Click Gaming** é um jogo do gênero _clicker_ desenvolvido com **React** e **Vite**. O objetivo é clicar freneticamente para acumular pontos, comprar melhorias (upgrades) de batata e desbloquear itens passivos que geram cliques automáticos.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes ferramentas:

- **React 18** (Hooks, Context API, Reducers)
- **Vite** (Build tool rápida)
- **CSS Modules** (Estilização encapsulada e responsiva)
- **React Icons** (Biblioteca de ícones)
- **React Router Dom** (Navegação entre páginas)

## 🛠️ Funcionalidades

- **Mecânica de Click**: Sistema de pontos baseado no valor acumulado por clique.
- **Loja de Upgrades**: Diferentes categorias de itens.
- **Gerenciamento de Estado**: Uso de `useReducer` e `Context API` para manter a lógica de cliques e compras síncrona em toda a aplicação.
- **Responsividade**: Não tem.
- **Identidade Visual**: Estética voltada para games com paleta de cores escuras e laranja neon.

## 📦 Como rodar o projeto

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/NowhereSA/AmazinClickGaming.git
   ```

   1.1 Entre na pasta:

   ```bash
   cd amazingclickgaming
   ```

   1.2 Instale as dependências:

   ```bash
   npm install
   ```

   1.3 Inicie o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```
